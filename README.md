# toStringFormatter

toStringFormatter can format Java toString() output (generated by Eclipse) in a more human readable form.

## Examples

### Basic object

```
Book [title=The Hitchhiker's Guide to the Galaxy, publishedYear=1979-10-12]
```

Become

```
Book [
    title=The Hitchhiker's Guide to the Galaxy,
    publishedYear=1979-10-12
]
```

### More complexe structure

```
Author [lastname=Douglas, firstname=Adams, birth=1952-03-11, bookWriten=[Book [title=The Hitchhiker's Guide to the Galaxy, publishedYear=1979-10-12], Book [title=The Restaurant at the End of the Universe, publishedYear=1980-10-01], Book [title=Life, the Universe and Everything, publishedYear=1982-08-01], Book [title=So Long, and Thanks for All the Fish, publishedYear=1984-11-01], Book [title=Mostly Harmless, publishedYear=1992-01-01]]]
```

Become

```
Author [
    lastname=Douglas,
    firstname=Adams,
    birth=1952-03-11,
    bookWriten=[
        Book [
            title=The Hitchhiker's Guide to the Galaxy,
            publishedYear=1979-10-12
        ],
        Book [
            title=The Restaurant at the End of the Universe,
            publishedYear=1980-10-01
        ],
        Book [
            title=Life,the Universe and Everything,
            publishedYear=1982-08-01
        ],
        Book [
            title=So Long,and Thanks for All the Fish,
            publishedYear=1984-11-01
        ],
        Book [
            title=Mostly Harmless,
            publishedYear=1992-01-01
        ]
    ]
]
```

### Example with Map

```
BeanThree [propMap={beanTwo1=BeanTwo [propBeanTwo=[BeanOne [propStr1=str1, propInt1=42], BeanOne [propStr1=str2, propInt1=43]]], beanTwo2=BeanTwo [propBeanTwo=[BeanOne [propStr1=str3, propInt1=44], BeanOne [propStr1=str4, propInt1=45]]]}]
```

Become

```
BeanThree [
    propMap={
        beanTwo1=BeanTwo [
            propBeanTwo=[
                BeanOne [
                    propStr1=str1,
                    propInt1=42
                ],
                BeanOne [
                    propStr1=str2,
                    propInt1=43
                ]
            ]
        ],
        beanTwo2=BeanTwo [
            propBeanTwo=[
                BeanOne [
                    propStr1=str3,
                    propInt1=44
                ],
                BeanOne [
                    propStr1=str4,
                    propInt1=45
                ]
            ]
        ]
    }
]
```
